About the PeptoVarDB project

PeptoVarDB is an on-line database of annotated peptide based on human genome annotation and public data of human genome common variants created with PeptoVar program in the pupulation mode. PeptoVar (Peptides of Variations) is a program for annotation of genomic variants in protein coding genes and generation of annotated peptides (open-projects.github.io/PeptoVar/). The main idea of PeptoVar is implementation of the potential compound effects of combining alternate alleles across multiple variant loci in sequence translation. PeptoVar is helpful for in-silico prediction of new minor histocompatibility antigen (MiHa), simulation of stem sells transplantation or as peptide database generator for shotgun proteomic studies of immunopeptidome.
The current release of PeptoVarDB based on RefSeq Annotation release 108 (www.ncbi.nlm.nih.gov/refseq/) and dbSNP Build 151 (www.ncbi.nlm.nih.gov/projects/SNP/). Only genome variants with Allele Frequency (AF) > 0.01 was used as the input data for the PeptoVar program. The program was started with the next parameters:
PeptoVar.py -tagaf CAF -minaf 0.01 -peptlen 8 9 10 11 12 -var nonsyn -indir ./input
